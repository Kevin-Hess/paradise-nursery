# Paradise Nursery

Paradise Nursery is a React and Redux single-page shopping application for browsing houseplants, adding them to a shopping cart, and adjusting quantities before checkout.

## Project Name
Paradise Nursery

## Description
This application allows users to:
- View a landing page introducing the Paradise Nursery company
- Browse a product listing page with houseplants organized into categories
- Add plants to a shopping cart
- View and manage the shopping cart, including increasing, decreasing, and deleting items
- See a dynamically updating cart total in the header

## Tech Stack
- React
- Redux Toolkit
- React Router
- Vite

## Pages
1. Landing Page - company introduction and Get Started button
2. Product Listing Page - houseplants grouped by category with Add to Cart functionality
3. Shopping Cart Page - cart contents, quantity controls, delete, checkout, and continue shopping
